# **App Name**: SuiCare

## Core Features:

- zkLogin Integration: Enable user onboarding and wallet creation using Google accounts via Slush Wallet, simplifying access to the Sui blockchain.
- Gasless Donations: Implement a backend "Gas Station" to sponsor user transactions, removing the need for users to pay gas fees directly.
- Verified Victim Registry: Maintain an on-chain registry where NGOs can vouch for victim addresses, ensuring donations are directed to verified recipients.
- Donation Claim Links: Allow NGOs to send donation "Claim Links" to victims via SMS/WhatsApp, enabling easy access to funds.
- Real-time Impact Feed: Display a live ticker of real-time donations using on-chain events, providing transparency and immediate feedback.
- Anti-Spam Gate: Enforce a minimum donation threshold to prevent spam and abuse of the gas sponsorship system.
- Impact Receipts: Automatically mint a non-transferable NFT to every donor, acting as a proof of donation and impact.

## Style Guidelines:

- Primary color: Light green (#B2FF59) for a sense of transparency and trustworthiness.
- Background color: Very light green (#F0FFF0) to provide a soft and clean backdrop.
- Accent color: Yellow-green (#BFFF57) for interactive elements and key highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif font.
- Use minimalist icons with a focus on clarity and ease of understanding.
- Implement a mobile-first design with spatial hierarchy, using drawers for donation inputs to maintain a clean main screen.
- Employ dynamic progress bars with 'Spring' animations to update on-chain pool balances in real-time.
